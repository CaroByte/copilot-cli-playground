# Sample Architecture

## Overview

The playground is organized by language and content type so each area is easy to explore.

## Directory layout

- `js-samples/` holds small JavaScript examples.
- `py-samples/` holds small Python examples.
- `markdown-samples/` holds lightweight documentation.
- `tests/` is reserved for future automated tests.
- `config/` is reserved for configuration examples.
